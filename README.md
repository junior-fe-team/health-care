# Health-care-wep

## 실행 방법

```bash
$ yarn install
```

**클라이언트**

```bash
$ yarn dev
```

## 브랜치 전략

- feat/(example) (기능 개발 브랜치)
- dev (개발용 브랜치)
- release/(date) (배포용 브랜치)

각 기능을 개발할 때 `feat/example`과 같이 브랜치를 생성하여 작업후, dev 브랜치에서 합치고 release 브랜치에서 배포하는 전략을 세웠습니다.
